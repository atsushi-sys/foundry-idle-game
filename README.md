# Foundry Idle Game

半導体ファウンドリ経営放置ゲーム。

## 起動方法

1. `index.html` をスマホブラウザで開く（またはローカルサーバーで配信）。
2. 1秒ごとにゲームが進行し、材料供給・3工程(FAB)・売上が更新されます。
3. 状態は `localStorage` に保存され、再起動時は最大8時間分のオフライン進行を適用します。

## Factory View（上段）

- 画面上半分に固定の「Factory View」を表示。
- クリーンルーム内のコンベア、3工程ステーション（Litho/Etch/CMP）、流れるウェハを可視化。
- ボトルネック工程のステーションがハイライト。
- ノード切替（0.13um / 40nm / 7nm）でテーマカラーと背景演出が変化。
- Effects / SFX トグルは `localStorage` に保存されます。

## Dashboard（下段）

- 既存の管理UI（資金、材料、FAB、アップグレード、ノード移行）をスクロール可能な下段に配置。
- 1ファイル構成（`index.html`）のまま動作します。
