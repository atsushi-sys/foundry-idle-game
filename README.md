# Foundry Idle Game

半導体ファウンドリ経営放置ゲーム。

## MVPの起動方法

1. `index.html` をスマホブラウザで開く（またはローカルサーバーで配信）。
2. 1秒ごとにゲームが進行し、材料供給・3工程(FAB)・売上が更新されます。
3. 状態は自動で `localStorage` に保存され、再起動時は最大8時間分のオフライン収益が加算されます。

## 実装メモ

- 1ファイル構成: `index.html`（HTML / CSS / JavaScriptを同梱）
- ダークな工場UI（クリーンルーム/制御盤風）
- FABは `Litho / Etch / CMP` の3工程で、ボトルネック工程が総スループットを決定
- ノード移行: `0.13um → 40nm → 7nm`
